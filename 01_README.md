# Sales Performance Dashboard

## Project Overview
This project is an interactive Sales Performance Dashboard built using Microsoft Excel.

The dashboard helps analyze:
- Product Performance
- City-wise Revenue
- Salesperson Performance
- Total Revenue
- Total Orders
- Average Revenue

---

## Features

### KPI Cards
- Total Revenue
- Total Orders
- Average Revenue

### Pivot Tables
- Revenue by Product
- Revenue by City
- Revenue by Salesperson

### Pivot Charts
- Product Revenue Analysis
- City Revenue Analysis
- Salesperson Performance Analysis

### Slicers
- Product Filter
- City Filter

---

## Business Insights

- Laptop generated the highest revenue.
- Delhi was the top-performing city.
- Top-performing salesperson can be rewarded and used as a mentor for other salespersons.
- Low-performing regions can be targeted for business growth.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Validation
